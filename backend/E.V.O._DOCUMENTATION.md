Cross-dimensional patterns that reveal system-level intelligence:

### Negative Emergence (Cascading Failures)

## Database Schema

### evo_analysis_cache
Stores complete synthesis results for 24-hour caching.
```sql
- id (SERIAL PRIMARY KEY)
- property_id (INTEGER)
- analysis_type (VARCHAR) -- 'full_synthesis', 'substrate', etc.
- dimensional_data (JSONB)
- emergence_patterns (JSONB)
- system_intelligence (JSONB)
- analyzed_at (TIMESTAMP)
- expires_at (TIMESTAMP)
```

### dimensional_health_history
Tracks health scores over time for trending.
```sql
- id (SERIAL PRIMARY KEY)
- property_id (INTEGER)
- dimension_name (VARCHAR)
- health_score (INTEGER) -- 0-100
- metrics (JSONB)
- measured_at (TIMESTAMP)
```

### emergence_patterns_history
Records detected patterns for pattern evolution tracking.
```sql
- id (SERIAL PRIMARY KEY)
- property_id (INTEGER)
- pattern_type (VARCHAR)
- severity (VARCHAR)
- dimensions_affected (TEXT[])
- insight (TEXT)
- detected_at (TIMESTAMP)
```

### evo_recommendations
Stores action recommendations with status tracking.
```sql
- id (SERIAL PRIMARY KEY)
- property_id (INTEGER)
- recommendation_type (VARCHAR)
- priority (INTEGER)
- action_description (TEXT)
- dimensions_affected (TEXT[])
- estimated_impact (JSONB)
- status (VARCHAR) -- 'pending', 'in_progress', 'completed'

```

## Transmutation Score Algorithm

The transmutation score predicts likelihood and impact of ranking improvement.

```javascript
transmutationScore = (
  (100 - keywordDifficulty) * 0.3 +  // Base metal softness
  (21 - currentPosition) * 2.5 +      // Proximity to top 10
  (impressions / 1000) * 0.2 +        // Volume available
  (hasCatalyst ? 20 : 0)              // Structured data bonus
) / 100
```

**Range**: 0-100  
**70-100**: High potential (immediate action)  
**50-69**: Medium potential (queue for optimization)  
**0-49**: Low potential (monitor for changes)

## Troubleshooting

### Common Issues

**"No GSC data returned"**
- Verify user has access to property
- Check OAuth token validity
- Ensure property has recent data

**"Ahrefs backlinks empty"**
- MCP integration requires actual Ahrefs API calls
- Placeholder returns mock data currently
- Implement actual MCP tool calls for production

**"Synthesis taking too long"**
- Check GSC API rate limits
- Verify network connectivity
- Consider reducing rowLimit in queries
